# Gros Gradient

Gros Gradient builds production-grade artificial intelligence, data systems and web applications.

All projects are designed and built in-house from end to end, with a focus on real-world usage, system reliability and operational leverage.

The organization publishes selected open-source tools, experiments and infrastructure components across AI, data and web engineering.

---

## Focus areas

AI systems including RAG pipelines, semantic search and intelligent assistants  
Data systems including dashboards, analytics and internal decision tools  
Automation systems that reduce operational friction and repetitive work  
Web applications and platforms built for production environments  
Custom software designed around real operational constraints  

---

## Principles

Designed in-house  
Built in-house  
Deployed in-house  

Systems are optimized for clarity, maintainability and real usage rather than theoretical completeness.

---

## Open source

This organization shares production-inspired components, experimental systems and engineering building blocks used in real deployments.

Each repository reflects a focus on practical execution rather than conceptual prototypes.
